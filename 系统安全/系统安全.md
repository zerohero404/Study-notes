# 系统安全防御
## 1.常见攻击手段
- 拒绝服务（DOS）
  - 通过大量访问请求使服务器资源耗尽，无法向正常用户处理请求连接
  -常见解决手段：使用云服务商的流量清洗功能
  - SQL的使用
    - DDL - 数据定义语言 - create / drop / alter
    - DML - 数据操作语言 - insert / delete / update / select
    - DCL - 数据控制语言 - grant / revoke
